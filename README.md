# chessmultiplayer ![Dependency Status][daviddm-image]

> A multiplayer command line chess game.

## Installation

```sh
$ git clone https://github.com/elimerl/chesscli
$ cd chesscli
$ yarn install
$ tsc
# Start server
$ yarn start
# or start client
$ node dist/client.js
```
